
# この文書について

この文書は「Android Studioではじめる 簡単Androidアプリ開発」の内容を補足する目的で、Android Studioのインストールと初期設定について説明していました。

現在は「Android Studioセットアップガイド」として、Android Studioのインストールと初期設定のみに限定しています。
